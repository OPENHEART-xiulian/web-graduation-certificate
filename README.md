# 项目概述
自动化生成电子证书平台，支持将证件信息以Excel文件形式导入到系统，然后根据导入内容生成电子证件，且可对电子证件进行手动盖章。生成的电子证件支持扫描二维码预览、保存。
前端采用node版本18以上使用VUE3.0+iview3.0框架进行部署，后端采用node进行编程。

# 运行命令

安装依赖

npm run install

启动项目

npm run serve

打包项目

npm run build
